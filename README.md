# Net Audit Automation

Herramienta en Python para automatizar tareas de auditoría interna autorizada de red, con fines de reconocimiento controlado, inventario técnico y apoyo a informes profesionales.

## Aviso importante

Esta herramienta está diseñada exclusivamente para auditorías de seguridad con autorización expresa del propietario del sistema o de la red. No debe utilizarse fuera de un alcance formalmente autorizado.

## Objetivos iniciales

- Descubrir hosts activos
- Enumerar puertos y servicios
- Normalizar resultados
- Exportar evidencias técnicas
- Facilitar la elaboración de informes

## Estado del proyecto

Versión inicial en desarrollo.

## Roadmap resumido

- Base del proyecto y CLI
- Integración con Nmap
- Parseo de resultados
- Exportación a JSON/CSV/Markdown
- Priorización heurística básica

## Entorno previsto

- Windows 11 + VirtualBox
- Kali Linux como entorno de ejecución
- Python 3.11+
- Nmap instalado en Kali

## Uso autorizado

Solo para evaluación defensiva y auditoría técnica autorizada.
